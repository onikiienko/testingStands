1. Установить VirtualBox

Для установки виртуальной windows машины на компьютер с Ubuntu, необходимо установить VirtualBox, для этого можно установить с официального сайта, либо выполнить следующую команду в терминале.

sudo apt-get install virtualbox

Официальный сайт

3. Переносим образы в VirtualBox

  • Запускаем VirtualBox.
  • Закрываем VirtualBox:) После этого у нас должна в домашней директории появиться папка "VirtualBox VMs". Если папка не появилась- создаем её вручную.
  • Переносим в неё наши образы. НЕ АРХИВЫ.
  • Заходим в любой из них, находим там файл формата vbox и запускаем его. Открылся VirtualBox.
  • В левой панели кликаем на виртуальную машины.
  • Соглашаемся со всеми окнами, которые всплывали.
  • Готово!
